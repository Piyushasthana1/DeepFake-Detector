About This Project

This DeepFake Detection System is an AI-powered web application designed to identify manipulated or synthetic videos using advanced deep learning techniques.

Technologies Used

  • Convolutional Neural Networks (CNN - EfficientNet)
  • LSTM for temporal sequence learning
  • Attention Mechanism for feature focus
  • Flask Web Framework
  • Google Cloud Deployment

Key Features

  • Real-time video analysis
  • Secure user authentication
  • High accuracy deepfake detection (~95%+)
  • Interactive and responsive UI

Project Goal

The goal of this project is to combat misinformation and enhance digital media trust by providing a reliable tool to detect deepfake content.
